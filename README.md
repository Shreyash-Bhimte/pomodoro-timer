# Pomodoro Timer

A minimal, distraction-free Pomodoro timer built with vanilla HTML, CSS, and JavaScript.

**Live demo:** (coming after deployment)

## Features
- 25/5/15 work-break cycle
- Auto session switching
- Web Audio API beep (no audio files)
- Custom durations saved to localStorage
- Mobile responsive

## Tech
Pure HTML · CSS Variables · Vanilla JS · Web Audio API · localStorage · GitHub Pages
